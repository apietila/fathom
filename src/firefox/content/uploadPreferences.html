Fathom is research project with the goal to improve the troubleshooting experience of Internet users. We are working on tools to assist users diagnose problems in their Web browsing sessions and suggest mechanisms to improve performance issues. To achieve this goal the first step is to collect more information about how client systems and Web browsers are configured and used.

To understand how various browser and system configurations affect Web browsing, we collect the following set of measurements that are recorded on the end host system. These measurements can be uploaded automatically or manually to our servers. However, you can chose to skip any of them when running Fathom. The uploaded data is not shared with any third party.

Thank you for your cooperation.
